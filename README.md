# Student Mental Health Tracker

## Overview

Student Mental Health Tracker is an AI-powered web application that helps students monitor their emotional well-being through mood tracking and journal analysis.

The application uses Natural Language Processing (NLP) to analyze journal entries and classify emotions as Positive, Negative, or Neutral. It also provides wellness suggestions and visual analytics.

## Features

- Mood Tracking
- Daily Journal Entries
- AI Sentiment Analysis using TextBlob
- Personalized Wellness Suggestions
- Interactive Dashboard
- Mood Distribution Charts
- PDF Report Generation
- SQLite Database Storage

## Technologies Used

- Python
- Flask
- SQLite
- TextBlob
- Bootstrap 5
- Chart.js
- ReportLab
